# House Hill Star — Photo Assets

Drop photos into the subfolder that matches the subject. Naming convention keeps everything searchable and helps with auto-pick on the website.

## Folders

| Folder | Use for |
|--------|---------|
| `interior/` | Bedroom, living room, kitchen, bathroom |
| `terrace/` | Outdoor terrace, sea-view deck, balcony |
| `garden/` | Private garden, plants, outdoor furniture |
| `views/` | Bay of Kotor, Portonovi, horizon shots |
| `exterior/` | House facade, approach, hillside context |
| `lifestyle/` | Wine on table, breakfast spread, sunset moment |

## Naming convention

```
{subject}-{time-of-day}-{seq}.{ext}
```

Examples:
- `terrace-sunset-01.jpg`
- `interior-bedroom-day-02.jpg`
- `views-bay-twilight-01.jpg`
- `garden-day-03.jpg`

**Time-of-day** values: `day`, `golden`, `sunset`, `twilight`, `night`, `dawn`

## Recommended sizes

- Original / source: keep as-is (we'll resize at build time)
- Minimum width: 1600px for hero candidates
- Format: JPEG (`.jpg`) for photos, WebP if you have it

## Hero candidates

For the Hillside Twilight homepage, ideal hero photos are:
1. Terrace at sunset or twilight (gold + indigo natural lighting)
2. Drone shot of house on the hill above Portonovi
3. Bay of Kotor view from the terrace at any hour

If those don't exist yet, we'll color-grade existing daytime photos and lean on typography for atmosphere.
